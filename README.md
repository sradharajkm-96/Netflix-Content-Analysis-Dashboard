# 🎬 Netflix Content Analysis Dashboard

An interactive dashboard project created using *Microsoft Excel* and *Power BI* to analyze Netflix Movies and TV Shows.

---

## 📌 Project Overview

This project transforms raw Netflix content data into meaningful insights through data cleaning, analysis, and interactive dashboards.

The dashboard analyzes Netflix content based on:

- Movies and TV Shows distribution
- Popular genres and categories
- Content trends over the years
- Ratings and audience suitability
- Top contributing countries
- Missing-value handling and data cleaning

---

## 🎯 Project Objectives

- Analyze the distribution of Movies and TV Shows on Netflix.
- Identify the most popular genres and content categories.
- Study content trends over the years.
- Analyze content ratings and audience suitability.
- Identify countries contributing the most content to Netflix.
- Handle missing values and clean the dataset.
- Create interactive dashboards and visualizations in Excel and Power BI.

---

- DAX
